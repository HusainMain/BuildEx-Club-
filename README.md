## 📌 Core Pillars

### ⚡️ High-Performance UI
Built with an elite design language emphasizing aesthetics and speed. The interfaces are fluid, cinematic, and heavily responsive.
- **Perspective Grids:** Deep spatial orientation for backgrounds.
- **Glassmorphism:** Layered UI elements built with intelligent opacity and frosted glass.
- **Aurora Lighting:** Reactive ambient gradient orbs and motion elements.

### 📡 Hardware Integration
Bridging software and physical presence through deep hardware synergy.
- **Haptic Feedback:** Engine vibrates upon successful QR scanning for tactile confirmation.
- **QR Operations:** HTML5 real-time continuous scanning, skipping third-party apps entirely.
- **Flashlight Control:** Hardware-level torch toggle built directly into the scanner interface for low-light halls.

### 🏗️ System Architecture
Secure, scalable, and instantaneous.
- **Role-Based Routing:** Strict application-level access controls separating Students and Admin Staff scopes.
- **3-State Event Lifecycle:** Intelligent algorithmic routing categorizing operations automatically by *Upcoming*, *Live*, and *History*.
- **Live Sync:** Postgres connectivity instantly updates registration statistics across client dashboards via Supabase.

## 🚀 Installation & Setup

1. **Clone the Repository**
   ```bash
   git clone https://github.com/HusainMain/BuildEx-Club-.git
   cd build-ex-platform
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment Variables**
   Create a local `.env` file at the root of the project to connect your Supabase infrastructure. 
   *(Note: For security reasons, the `.env` file is untracked and should never be pushed to version control.)*
   ```ini
   # .env
   VITE_SUPABASE_URL=your_supabase_project_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

4. **Launch Development Server**
   ```bash
   npm run dev
   ```

## 📱 Mobile Experience (PWA)
Build Ex operates identically to a native application.
- **Add to Home Screen (A2HS):** PWA manifest integration perfectly sets the application to launch standalone from your mobile home screen.
- **Native Portrait Lock:** Engine locked statically to portrait via `manifest.json`, eliminating UI overflow and rotation breakage.
- **Anti-Zoom Security:** Disabled viewport scaling ensures rapid double taps won't trigger unwanted browser zooming.
